# Quick Start

## Setup

Windows:
```bat
setup.bat
```

Linux/Mac:
```bash
bash setup.sh
```

## Run Dashboard

Windows:
```bat
start_dashboard.bat
```

Linux/Mac:
```bash
bash start_dashboard.sh
```

Streamlit will print a local URL, usually `http://localhost:8501`.

Dashboard code lives in `spectra_dashboard/main.py`.
